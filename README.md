# USB Tools

A PowerShell toolkit for USB device management and utilities.

## Description

This project provides PowerShell scripts and tools for working with USB devices.

## Requirements

- Windows PowerShell 5.1 or PowerShell 7+
- Administrator privileges (for some operations)

## Usage

```powershell
# Basic usage example
.\usbtools.ps1
```
